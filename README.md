# piirremcon
Respberry Pi IR remote controller

## What's this?

This software make Raspberry Pi a universal IR remote controller
for appliances.

## Features

  - No special library

## Working targets

  - SHARP TV (Aquos)
  - SONY TV (BRAVIA)
  - TOSHIBA TV (REGZA)
  - NEC projector
  - EPSON projector
  - SONY Blu-ray player
